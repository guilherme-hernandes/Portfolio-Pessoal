# Portfólio Pessoal

Projeto de portfólio pessoal criado durante o curso "Dividindo para Somar - Front-End - Criando Interfaces Web Responsivas do Zero".

Descrição
---------
Este repositório contém uma página estática de portfólio construída com HTML e CSS. É um projeto simples para praticar estruturação de páginas, estilos responsivos e organização de arquivos.

Estrutura do repositório
------------------------
- `index.html` — arquivo principal do site.
- `css/` — pasta com arquivos de estilo.
	- `main.css` — estilos principais do projeto.

Como executar
-------------
1. Abra o arquivo `index.html` diretamente em um navegador (duplo-clique).
2. Para desenvolvimento com recarga automática, use a extensão Live Server do VS Code e abra o projeto com ela.

Tecnologias
----------
- HTML
- CSS
