# Portfólio Pessoal

Pequeno site de portfólio pessoal criado como exercício do curso "Dividindo para Somar - Front-End: Criando Interfaces Web Responsivas do Zero".

Visão Geral
-----------
Site estático construído com HTML e CSS para apresentar informações pessoais, projetos e contatos.

Estrutura do Repositório
------------------------
- `index.html` — página principal.
- `css/main.css` — estilos do projeto.
- `js/main.js` — comportamentos e melhorias de acessibilidade.

Como Visualizar
---------------
1. Abrir `index.html` diretamente no navegador (duplo-clique) para visualização básica.
2. Para desenvolvimento com recarga automática, usar a extensão Live Server do VS Code.

Personalização Rápida
---------------------
- Para calcular a idade automaticamente, definir o atributo `data-birth="YYYY-MM-DD"` no elemento com a classe `.age` ou no elemento `<body>`.
- Substituir os textos de exemplo em `index.html` por seus dados (nome, biografia, links de projetos).

Desenvolvimento
---------------
- Projeto estático — não requer servidor para rodar localmente.
- Recomenda-se ter um linter/formatador se for estender o código (ex.: `prettier`, `eslint`).

Boas Práticas (Git)
-------------------
É recomendado incluir um arquivo `.gitignore` para evitar comitar arquivos de build, dependências ou arquivos do sistema/IDE. Neste repositório já foi adicionado um `.gitignore` com itens comuns.
